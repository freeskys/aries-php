AriesPHP

AriesPHP is a Microframework to create MVC based website in PHP. AriesPHP designed to be fast, secure, and easy. We work hard to make AriesPHP as easy as we can, so everyone can make website without pain.

Download latest version of AriesPHP version 1.0.4: https://www.box.com/s/bonyv65zonher52lko55
